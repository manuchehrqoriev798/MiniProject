# English Assessment Platform

A web-based assessment platform for 9th grade students to take English proficiency exams and receive instant PDF reports.

---

## 🎯 Project Goal

**Learn modern full-stack development by building a real, production-ready application.**

This project teaches:

- **Backend**: FastAPI, PostgreSQL, JWT authentication, PDF generation
- **Frontend**: React, form handling, protected routes, API integration
- **Full-Stack**: Building complete user journeys from registration to reports
- **Teamwork**: Git workflow, code reviews, API contracts, sprint planning

**After this project, the team will be ready to build complex, scalable web applications.**

---

## 🚀 What We're Building (2-Week MVP)

### For Students

- Register with name and phone number
- Take 15-question English exam (20 minutes, paginated)
- Receive PDF report with score and percentage

### For Teachers

- Secure login
- Create, edit, and delete exam questions
- Manage question bank

---

## 👥 Team

- **3 Frontend Developers** (React)
- **1 Backend Developer** (FastAPI + PostgreSQL)
- **1 Team Lead** (Code reviews, integration, deployment)
- **1 Project Manager** (Scope, timeline, stakeholder communication)

---

## 🛠️ Tech Stack

| Layer              | Technology   | Why                             |
| ------------------ | ------------ | ------------------------------- |
| **Frontend**       | React + Vite | Modern, fast, component-based   |
| **Backend**        | FastAPI      | Modern Python, auto-docs, fast  |
| **Database**       | PostgreSQL   | Industry-standard SQL, powerful |
| **Authentication** | JWT          | Stateless, secure, scalable     |
| **Reports**        | ReportLab    | PDF generation in Python        |
